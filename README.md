# Official ReCord Theme

## Overview

Official ReCord Theme is a modern glass-based interface designed to enhance the visual experience of ReCord. It introduces a refined aesthetic built around clarity, depth, and subtle transparency, while maintaining usability and performance.

The theme focuses on clean layering, smooth visual hierarchy, and a consistent color system to deliver a cohesive and polished user interface.

---

## Features

* Frosted glass interface with backdrop blur
* Consistent and centralized color system
* Soft accent highlights for interactions
* Refined channel, chat, and sidebar styling
* Minimalist scrollbars and UI elements
* Smooth hover and transition effects
* Optimized layout spacing and readability

---

## Installation

1. Download the theme file.
2. Move the file into your themes directory.
3. Enable the theme through your client.

---

## Configuration

The theme uses a structured variable system defined in `:root`. You can easily customize:

* Accent color
* Background opacity
* Blur strength
* Border intensity
* Radius and spacing

Example:

```css
:root {
  --rc-accent: rgb(94, 129, 172);
  --rc-blur: 16px;
}
```

---

## Compatibility

* Designed for ReCord-compatible clients
* Works with Glass-based rendering environments
* Supports both light and dark UI contexts where applicable

---

## Performance

The theme is optimized for smooth rendering. However, blur effects may vary depending on system performance and hardware acceleration.
